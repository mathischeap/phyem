Download#

We recommend you to download the source code from the GitHub page, PHYEM (GitHub). Or, run the following git command to clone the package to a local repository,

git clone https://github.com/mathischeap/phyem.git

In the future, you can also get PHYEM through the Python Package Index (PyPI) by running the following command,

pip install  *****
