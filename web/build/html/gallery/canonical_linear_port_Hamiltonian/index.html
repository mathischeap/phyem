Canonical linear port Hamiltonian#

Here we demonstrate how to use PHYEM to solve the canonical linear port Hamiltonian problem in three dimensions.

In \(\mathbb{R}^3\), a canonical form the linear port-Hamiltonian is written as

\[\begin{split}\begin{equation}\left\lbrace \begin{aligned} \partial_t \alpha^3 & = - \mathrm{d}\beta^2\\ \partial_t \beta^2 & = \mathrm{d}^{\ast} \alpha^3 \end{aligned}\right. \end{equation}\end{split}\]

It is solved with manufactured solutions. The program is

canonical_linear_pH_3d_periodic_manufactured_test(degree, K, num_steps)[source]#
Parameters:
degreeint

The degree of mimetic spectral elements.

Kint

We use \(K*K*K\) uniform elements.

num_stepsint

How many steps we want to run? Each time step is \(\Delta t = 0.01\). Maximum num_steps is 100.

Returns:
a3_L2_errorfloat

\(L^2\)-error of solution \(\alpha_h^3\) at the last time step.

b2_L2_errorfloat

\(L^2\)-error of solution \(\beta_h^2\) at the last time step.

It solves the canonical linear port Hamiltonian problem in a periodic unit cube, \(\Omega:=[0,1]^3\). The time step is \(\Delta t = 0.01\).

Examples#

If we solve the problem with mimetic spectral element of degree 1 on a mesh of \(3*3*3\) elements for only one time step, we can do

>>> errors1 = canonical_linear_pH_3d_periodic_manufactured_test(1, 3, 1)

To check the \(L^2\)-error of solution \(\alpha_h^3\) (at \(t=0.01\) since we only solve for one time step), do

>>> errors1[0]  
5.2...

You may do \(ph\) refinements to decrease the error or run more time-steps to check the error after longer iterations.
